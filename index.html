What is PrBoom-Plus-Enhanced ?

PrBoom-Plus-Enhanced is a source port for the old school video game in the category of first person shooter named DOOM relesed 1993 by ID Software and it was released open-source and under GPL-license in 1997, by John CARMACK.

So in the DOOM tree of source ports, PrBoom-Plus-Enhanced is after DSDA DOOM. PrBooM-Plus-ENHANCED takes features from a lot of source ports like PrBoom, Chocolate Doom, Crispy Doom, MBF even some features of FastDOOM.

PrBoom-Plus-Enhanced can be qualified as a BOOM based source port.

What are the purposes of PrBoom-Plus-Enhanced ?

The goals of PrBoom-Plus-Enhanced are :

        
        -Learning to work with people together
        
        -Learning how to make a homebrew
        
        -Learning how github works
        
        -Learning how to use a repository
        
        -Learning how make a website
        
        -Learning how to make a communty around the project
        
        -Make the ultimate port of DOOM on the Nintendo 3DS family systems Old and New models
        
        -Exclusive to the Nintendo 3DS version learn how to add old features from older homebrews DS, 3DS...
        (DSDOOM, PRBOOM 3DS, DSCHOCOLATE DOOM...)
